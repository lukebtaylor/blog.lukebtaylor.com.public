Keepass Desktop Setup

Published: November 23, 2018 Estimated reading time: ~1 minutes

Use KeePass for password management.

It offers syncing with cloud storage providers, multiple plug-ins, and mobile apps.

Install KeePass

  1. Install using Chocolatey choco install keepass -y
  2. Or install KeePass from here https://keepass.info/
  3. Create a database in a cloud folder like NextCloud or Dropbox with a keyfile and a password
  4. Save the password in a very safe place
  5. Save the keyfile in a very safe place
  6. If you lose or forget the password or keyfile, the password database will not be able to be opened

Install the Kee browser add-on for FireFox and Chrome

  1. Download the KeePassRPC Plugin from the Keepass Plugins
  2. Copy the .plgx file to the C:\Program Files (x86)\KeePass Password Safe 2\Plugins folder
  3. Restart KeePass
  4. Install the Kee plugin from the MMozilla Addon Center
  5. Follow the Kee Installation Instructions for additional help.
  6. Walk through the Kee Tutorial for more assistance
